# Monte_Carlo_proj
Independent Monte Carlo study within a CAPM framework


**Method**

Conducted a 10,000 replication Monte Carlo simulation within a CAPM framework to validate that when samples are large they tend to the normal distribution and see if this held true within a CAPM framework.

**Sections**

This was tested by varying the sample sizes
Varying the Betas within the CAPM formula 
Repeating the Monte Carlo simulation for 10,000 replications 
Measuring and storing the mean value for each replication to find the distribution of sample means across samples 

**Graphs** 

All plots were made for each sample size and Beta values

**Robustness/validation checks**

The code was replicated in R,EViews and Python to ensure consistency and validaty of the results and findings as well as test for replication.
